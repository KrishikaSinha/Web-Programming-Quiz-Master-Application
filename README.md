# 🎯 Quiz Master Application

## 📌 Description
Quiz Master is a full-stack web application developed using Flask that allows users to attempt multiple-choice questions and view their score instantly with feedback.

---

## 🚀 Features

### ✅ Core Features
- Flask-based backend application
- 5 Multiple Choice Questions (MCQs)
- Separate pages: Home, Quiz, Result
- Automatic score calculation
- Feedback based on performance

### ⭐ Bonus Features
- Restart Quiz option
- Timer for quiz
- Negative marking system
- Shuffle questions

---

## 🛠️ Tech Stack
- Frontend: HTML, CSS
- Backend: Python (Flask)

## ⚙️ How to Run

1. Install Flask  
pip install flask  

2. Run the app  
python app.py  

3. Open in browser  
http://127.0.0.1:5000/  

---

## 📊 Working
- User starts quiz from home page  
- Answers MCQs  
- Submits the form  
- System evaluates answers  
- Displays score and feedback  

---

## 📈 Outcome
This project demonstrates integration of frontend and backend technologies to build a functional and interactive web application using Flask.
